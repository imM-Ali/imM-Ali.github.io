What is HTML?

Html stands for Hyper Text Markup Language
it is a language used to write the structure
of a webpage. It uses tags to wrap elements so that
they could be understood and diplayed by the browser correctly.

Necessary tags for all webpages:

<html>

The whole webpage needs to be wrapped inside html tags.

<head>

All linked elements are wrapped inside the head tag. Head tag is at the start of the html document.

<body>

All content of the website is written inside the body tag. Body tag follows head tag.

Basic layout of a webpage

<!DOCTYPE html>
<head>
Metadata goes here
For Example, Title, CSS & links
<title> My example </title>
</head>
<body>
Content goes here
</body>
</html>
