What is Javascript?

Javascript is the most used scripting language. It is used to implement animative features that are triggered on certain events e.g clicks/hover/page-load etc.

Advantages of Javascript:

Rich Interface

Javascript enables us to create feature rich and appealing websites.

Light Weight

Javascript is extremly lightweight as it runs instantly on the browser itself.

Easy to learn

Javascript supports Object Oriented Programming hence is relatively easy to learn.

Basic Javascript

*Includes bootstrap classes
$("#Button").on('click',function(){
if($("#Icon").hasClass("fas fa-lightbulb")) {

$("#Icon").removeClass("fas fa-lightbulb").addClass("fas fa-lightbulb-on").addClass("text-warning");
$(this).text('Turn Off');
}
else {

$("#Icon").removeClass("fas fa-lightbulb-on").addClass("fas fa-lightbulb").removeClass("text-warning");
$(this).text('Turn On');
}
});
